Basic HTML Tags

by Matthew Cheng

Welcome! Below you will find a table consisting of some of the basic HTML tags I find the most helpful. Enjoy!

Tag Name Formatted Tag Description
HTML <html> This tag is the parent to all of the other tags in the text editor document
Head <head> This tag holds all of the information about the website that is not actually visible on the webpage
Title <title> This tag allows you to specify the title of the webpage on the tab
Link <link> This tag allows you to link another text editor, such as CSS, to your HTML docment
Body <body> This tag contains all of the visible information on the webpage
Divider <div> This tag creates a divider to separate different sections of the webpage
Paragraph <p> This tag creates a paragraph using the contents sandwiched between the opening and closing tags
Header <h1-h6> These tags create headers on the webpage based on the number of the header (largest-1, smallest-6)
Navigation <nav> This tag is used to create Navigation Blocks to allow users to travel around the webpage
Anchor <a> This tag is used to turn text into an anchor for links to other webpages
Unordered List <ul> This tag is used to create a bulleted list
Ordered List <ol> This tag is used to create a list with numbers instead of bullets
List Item <li> This tag is used to create the actual information present in the ordered or unordered list (actual bullets)
Emphasis <em> This tag causes the contained string to be put in italics
Strong <strong> This tag causes the contained string to be bolded
Image <img> This tag is used to insert an image into the webpage through link or local image file
Video <video> This tag is used to insert a video into the webpage similar to the img tag
Comment <!--text--> A comment can be made by writing the comment where 'text' is located. This comment has no affect on the actual coding
Break <br> This tag can be used to create a break in any string on the webpage

Thank you for reading my short guide. If you have any questions, or would like to learn more tags, check out other good resources, like GeeksforGeeks.
